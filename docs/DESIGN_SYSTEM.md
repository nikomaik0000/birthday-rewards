# Birthday Rewards - Design System

## Philosophy

The website should feel simple, clean and lightweight.

Avoid unnecessary decorations.

Content readability is always more important than visual effects.

---

# Border Radius

Cards
- rounded-xl (16px)

Buttons
- rounded-lg (12px)

Inputs
- rounded-lg (12px)

Dialogs
- rounded-2xl

---

# Shadows

Use only one soft shadow style.

Avoid heavy shadows.

---

# Spacing

Use an 8px spacing system.

Common spacing:

- 8
- 12
- 16
- 24
- 32

---

# Typography

Page Title

Section Title

Body

Caption

Keep typography consistent.

---

# Icons

Preferred:

Lucide Icons

Size:

16px

18px

20px

Do not mix icon styles.

---

# Colors

Use the existing light theme.

Do not introduce additional accent colors unless required.

---

# Components

Keep components visually consistent.

Avoid creating similar components with different styles.

Reuse existing UI components whenever possible.

---

# Mobile

Mobile should feel app-like.

Avoid horizontal scrolling.

Touch targets should be easy to tap.

---

# General Rules

Prefer simplicity.

Avoid feature bloat.

Keep the UI focused on birthday reward information.
